# Chat-Server

This repository is dedicated to learning how basic network programming is done and how to establish connections between multiple clients through a server.

### **The goals of this project are the following:**
1. [x] Send message from client to server and respond with acknowledgement.
2. [x] Keep session open between client and server until a specific input occurs.
3. [x] Send messages from various clients.
4. [ ] Redirect traffic from each client to all other clients.
5. [ ] Send confirmation of received messages by clients to original client.
6. [ ] Give each client a unique ID (with username) and store bank of ids on server.
7. [ ] Make conversations work based on petitions from client to client.
8. [ ] Store conversations in files.
9. [ ] Retrieve all messages stored when requested by client of a specific conversation.
10. [ ] GUI of chat conversations.
11. [ ] Establish acknowledgements of locally stored messages. Retrieve messages of only missing messages.
